AkadCampus
======

AkadCampus Java Library und Console zum exportieren der Termine als CSV


Console
=======

```
usage: java -jar akad-campus.jar
 -h,--help          Hilfe
 -p,--pass <arg>    Passwort
 -u,--user <arg>    Benutzername
 -version           Version

```

Ant Build Distribution
======================

```
ant dist
```
